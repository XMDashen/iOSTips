##UIDatePicker

###属性
```
1.初始化显示的date日期

date

2.设置最小日期

minimumDate

3.设置最大日期

maximumDate

4.设置日期的显示样式

datePickerMode

UIDatePickerModeTime 显示时间12小时制(上下午)

UIDatePickerModeDate 显示日期

UIDatePickerModeDateAndTime 显示日期和时间

UIDatePickerModeCountDownTimer 显示时间24小时制

5.分钟间隔值

minuteInterval

6.地区
locale
```
